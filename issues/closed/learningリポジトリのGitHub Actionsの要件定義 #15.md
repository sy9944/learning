## 今後の仕様
- issueごとに技術のラベルを付与する
- 全てのissueを検索して、ラベルごと(技術ごと)に集計する
- フロント、バックエンド等は区別しない

```

```

手動で更新する部分にはドキュメントへのリンクなどを編集する
